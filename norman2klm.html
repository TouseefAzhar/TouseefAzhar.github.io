Model Comparison

This table compares three important Human–Computer Interaction models. Norman’s model focuses on how users interact with systems, GOMS analyzes task structure for expert users, and KLM predicts execution time using low-level operators.

Comparison of Norman, GOMS, and KLM
Model Focus Purpose
Norman User actions and evaluation Explains interaction problems and usability issues
GOMS Goals and task decomposition Analyzes expert user performance
KLM Low-level operators Predicts task execution time
